功能亮点

Speakin 为输入而生

每一个细节都在消除语音与文字之间的距离

🎙

按住 Fn 即说

按住 Fn 键开始录音，松开自动完成转写并注入当前光标位置。全程无需切换窗口或点击任何按钮，在微信、备忘录、浏览器等任意应用中直接使用。

⚡

实时语音转写

基于阿里云 Paraformer-v2 实时语音识别引擎，支持语义断句、自动添加标点符号，并智能过滤"嗯""啊"等口语语气词，输出即是整洁的书面文字。

📍

光标位置跟随

录音时会在光标旁显示一个精致的胶囊指示器，实时展示录音波形和状态。通过 Accessibility API 精确定位，即使在 Electron 应用中也能准确跟随。

🌐

四种语言一键切换

支持简体中文、English、日本語、한국어 四种语言的语音识别。在菜单栏切换语言后，界面语言和识别语言同步切换，无需额外配置。

📋

剪贴板无损恢复

Speakin 通过模拟粘贴注入文字，但会在注入前备份剪贴板内容，注入后自动恢复。你的复制粘贴不会被打断，工作流完全无感。

🪶

极致轻量原生

纯 Swift + AppKit 原生开发，零第三方依赖。应用体积不到 3MB，内存占用极低，作为菜单栏常驻应用几乎无存在感，但随时待命。

开始使用

三分钟，从安装到输入

简单几步，Speakin 即刻就绪

⚠️ Speakin 尚未进行 Apple 公证签名，首次打开需手动授权。这不影响正常使用，按以下步骤操作即可。
1

下载并安装

下载 Speakin.dmg，打开后将 Speakin 拖入「应用程序」文件夹

2

解除安全限制

打开终端，运行以下命令：

xattr -cr /Applications/Speakin.app
3

授予权限

启动后按提示授予「辅助功能」和「麦克风」权限

4

配置 API Key

菜单栏图标 → 设置，填入 DashScope API Key（可在阿里云百炼平台免费申请）

5

开口就好

在任意输入框按住 Fn 说话，松开即输入。就这么简单。

多语言支持

你说什么语言，Speakin 就听什么语言

🇨🇳 简体中文
🇺🇸 English
🇯🇵 日本語
🇰🇷 한국어
